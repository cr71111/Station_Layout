📐 综合评分算法

每个场站的综合评分由 5个维度 加权计算得出，满分约 150 分。

综合分 = GPS热度分 + 距离分 + 电费单价分 + 用电度数分 + 电柜等级分

1. GPS热度分 0~50分

基于场站周边 GPS 定位点的密度计算。GPS 数据点越多，说明该区域用户需求越旺盛。

  • 对场站周边一定范围内的 GPS 点位进行统计
  • 采用分位数归一化，将原始 GPS 数量映射到 0~50 分
  • GPS 数量最多的场站获得满分 50 分

2. 距离分 0~50分

评估场站 1km / 2km / 3km 范围内现有电柜的分布密度。周边电柜越多，说明该区域换电需求越集中。

距离分 = (1km内高等级×6 + 1km内总柜×3) + (2km内高等级×2 + 2km内总柜×1) + (3km内高等级×1) × 权重系数
  • 1km 内电柜权重最高（高等级×6，总柜×3）
  • 2km 内电柜权重中等（高等级×2，总柜×1）
  • 3km 内电柜权重最低（高等级×1）
  • 高等级电柜（特高/高）的权重是中低等级电柜的 2 倍

3. 电费单价分 0~25分

场站周边电柜的平均电费单价越高，说明用自有场站替换后节省的电费成本越多，评分越高。

  • 统计 3km 范围内所有电柜的电费单价
  • 周边电费单价越高，替换后节省的成本越多，得分越高，满分 25 分

4. 用电度数分 0~25分

场站周边电柜的总用电量越大，说明该区域换电需求越旺盛。

  • 统计 3km 范围内所有电柜的用电度数总和
  • 用电量越大，得分越高，满分 25 分

5. 电柜等级分 0~20分

场站周边高等级电柜的占比越高，说明该区域换电基础设施越完善。

  • 等级分为：特高、高、中高、中、中低、低
  • 高等级电柜（特高/高）占比越高，得分越高
  • 满分 20 分

场站分类

分类条件说明
已选场站综合分 > 0 且 标记为"已选"优先推荐布局的场站
推荐布局综合分 > 0 且 未标记"已选"评分较高，建议关注
